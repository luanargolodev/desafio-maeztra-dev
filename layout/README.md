## 📁 Rodando o projeto

Clone o projeto

```bash
  git clone https://github.com/luanargolodev/desafio-maeztra-dev.git
```

Entre no diretório do projeto

```bash
  cd desafio-maeztra-dev
  cd layout
```

Instale as dependências necessárias

```bash
  pnpm i
```

Finalmente, podemos iniciar o servidor

```bash
  pnpm dev
```
